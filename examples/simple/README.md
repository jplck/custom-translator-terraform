# Simple Custom Translator Example

This example shows how to deploy a simple Azure Custom Translator service.

## Usage

1. Copy this directory to your workspace
2. Customize the `terraform.tfvars` file
3. Run:
   ```bash
   terraform init
   terraform plan
   terraform apply
   ```

## Files

- `main.tf` - Main configuration that calls the root module
- `terraform.tfvars` - Example variable values
- `outputs.tf` - Example outputs
